Блок навигации

  • Ширина 15em
  • Должен присутствовать цвет фона
  • Располагается слева
  • Блок навигации наполните контентом достаточным,что бы по высоте онначинал захватывать вторую статью,как показано в данном примере

Статья 1

Размер блока:
Занимает всю доступоную ширину
Должен присутствовать цвет фона
Во всех "статьях" сделайте отступ снизу от соседей равным 2em при помощи
свойства margin-bottom

Статья 2

Размер блока:
Занимает всю доступоную ширину
Должен присутствовать цвет фона
Во всех "статьях" сделайте отступ снизу от соседей равным 2em при помощи
свойства margin-bottom. Картинка вставлена при момощи тега,плавание
организованно по правому краю внутри данной статьи для этой картинки
Нижний блок,подвал.занимает всю доступную ширину.Высота определяется помещённым внутрьконтентом.
